Cheatsheet Docker

Docker Run

docker run [opciones] nombre-imagen

Este comando se utiliza para crear e iniciar un contenedor a partir de una imagen.

Docker Pull

docker pull nombre-imagen

Descarga una imagen de Docker Hub u otro repositorio de imágenes de contenedores.

Docker PS

docker ps

Muestra una lista de contenedores en ejecución.

Docker PS -a

docker ps -a

Muestra una lista de todos los contenedores, incluidos los que están detenidos.

Docker Images

docker images | head

Muestra las imágenes disponibles en el sistema.

Docker Start

docker start ID-contenedor

Inicia un contenedor detenido.

Docker Logs

docker logs ID-contenedor o nombre-contenedor

Muestra los registros de salida de un contenedor.

Docker Logs -f

docker logs -f ID-contenedor

Muestra los registros de salida de un contenedor en tiempo real.

Docker Exec

docker exec ID-contenedor comando

Ejecuta un comando en un contenedor en ejecución.

Docker Stop

docker stop ID-contenedor

Detiene un contenedor en ejecución.

Docker Build

docker build -t nombre-imagen .

Construye una imagen de Docker a partir de un Dockerfile en el directorio actual.

Docker Login

docker login

Inicia sesión en Docker Hub.

Docker Tag

docker tag ID-imagen nuevo-nombre

Etiqueta una imagen de Docker con un nuevo nombre o versión.

Docker Push

docker push nuevo-nombre

Sube una imagen de Docker a Docker Hub u otro repositorio.

Docker RM

docker rm ID-contenedor

Elimina uno o más contenedores.

Docker Image RM

docker image rm ID-imagen

Elimina una o más imágenes.
